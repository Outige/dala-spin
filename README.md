# Dala Spin

Static spin-to-win prize wheel for a trade-show tablet or display.

## Use

Open `index.html` in a browser, or upload these files to any static hosting provider:

- `index.html`
- `styles.css`
- `script.js`

Edit the clearly labelled `appConfig` object at the top of `script.js` to change prizes, odds, text, timing, sound, confetti, and result behavior. Set `soundEnabled` to `false` for a silent trade-show setup, or tune `soundVolume` for subtle booth audio.
